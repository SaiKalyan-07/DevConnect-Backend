# DevConnect Backend

DevConnect is a developer networking and collaboration platform designed to help developers connect, communicate, and expand their professional network.  
This repository contains the backend implementation built using **Node.js**, **Express.js**, **MongoDB**, and **Socket.io** for real-time communication.

---

## ⚙️ Tech Stack

- **Backend Framework:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ODM)  
- **Authentication:** JWT & Cookie-based Authentication  
- **Real-Time Communication:** Socket.io  
- **Security:** Bcrypt, JWT Validation  
- **Architecture:** Modular routing, middleware, controller, and utility structure  

---

## ✨ Features

- **User Authentication:** Secure signup, login, and logout using JWT-based sessions.  
- **Profile Management:** Retrieve and update user profiles.  
- **Connection Requests:** Send, accept, reject, or ignore connection requests.  
- **User Feed:** Paginated developer feed with profile and connection visibility.  
- **Chat System:** One-to-one real-time messaging using Socket.io with persistent chat storage.  
- **Middleware Authentication:** Token verification for protected routes and user actions.  
- **Validation Utilities:** Centralized validation utilities for clean and secure data handling.  
- **Scalable Architecture:** Organized and modular structure for routes, models, middlewares, controllers, and utilities, supporting easy future expansion.
