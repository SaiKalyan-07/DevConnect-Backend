# DevConnect Backend

DevConnect is a developer networking and collaboration platform designed to help developers connect, collaborate on projects, and expand their professional network.  
This repository contains the **backend implementation** built with **Node.js**, **Express.js**, and **MongoDB**.

---

## ⚙️ Tech Stack
- **Backend Framework:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ODM)  
- **Authentication:** JWT & Cookie-based Authentication  

---

## ✨ Features
- **User Authentication:** Sign up, login, and logout with secure JWT-based sessions.  
- **Profile Management:** View and edit user profiles.  
- **Connection Requests:** Send, accept, reject, or ignore connection requests between users.  
- **Feed System:** Paginated user feed displaying profiles and connections.  
- **Middleware Authentication:** Token verification for protected routes.  
- **Data Validation:** Input validation using utility functions for clean and secure data handling.  
- **Scalable Architecture:** Modular structure with separate routes, models, and middlewares.
